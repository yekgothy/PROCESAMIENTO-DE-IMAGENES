Manual de uso del ecosistema Eclipse

Sigue estas pautas para aprovechar cada módulo de contenido inmersivo: desde el escaneo inicial hasta las trivias y tableros de datos que viven fuera del visor.

Trivia

Rondas dinámicas

Activa la trivia desde el menú principal para desbloquear cuestionarios temáticos por estadio o país anfitrión.

  • Diez preguntas por sesión con retroalimentación inmediata.
  • Resultado final listo para compartir en redes.
  • Reinicia la experiencia sin salir del visor.
Historia

Contexto curado

Consulta la sección de historia para profundizar en hitos, figuras clave y cronologías exclusivas.

  • Contenido editorial sincronizado con cada marcador.
  • Textos disponibles en modo offline.
  • Accesos rápidos a enlaces de referencia.
Escaneo

Guía de captura

Para un reconocimiento óptimo, mantén el marcador dentro del cuadro guía y busca iluminación uniforme.

  • Evita reflejos directos o sombras duras.
  • Utiliza superficies planas para mayor estabilidad.
  • Explora la vista previa antes de activar efectos.
Datos

Paneles analíticos

El módulo de estadísticas muestra indicadores clave agregados por torneo, sede y jugador destacado.

  • Métricas en tiempo real alimentadas por la API oficial.
  • Filtros rápidos para comparar selecciones.
  • Exporta un resumen en PDF con un clic.
